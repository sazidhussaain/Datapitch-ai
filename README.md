# 📈 DataPitch AI Landing Page

A modern, responsive, and interactive landing page built for a fictional B2B tech startup, DataPitch AI. This project demonstrates clean UI/UX principles, dark-mode styling, and DOM manipulation using Vanilla JavaScript.

## 🚀 Features
* *Modern UI/UX:* A sleek "Deep Tech Blue" dark-mode aesthetic designed to build trust and feel enterprise-ready.
* *Responsive CSS Grid & Flexbox:* Fully mobile-responsive layout that adapts beautifully to any screen size.
* *Interactive AI Lead Score Simulator:* A custom-built JavaScript widget that simulates a predictive analytics engine, complete with loading states and number-counter animations.

## 🛠️ Tech Stack
* *HTML5:* Semantic markup for structure and accessibility.
* *CSS3:* Flexbox for precise alignment, Grid for responsive card layouts, and custom hover states.
* *Vanilla JavaScript:* Event listeners, setTimeout for artificial delays, and setInterval for dynamic UI number animations.
